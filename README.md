# Todo App
A simple todo app build using <b>express</b> and <b>ejs</b>

# Demo

# Setup
- Step 1:
```
$ git clone https://github.com/thapasunita/todo-app.git
$ cd todo-app
$ node index.js
```
- Step 2:

Open this in browser http://localhost:8000
